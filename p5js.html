Draft

Final

Using the programming language p5.js, I created an vector self-portrait. It was definitely a fun way to experiment! My
first attempt was free-handing everything and eyeballing all of the values I needed to create the image. For my real
attempt, I used inspiration from Seth Tobeck's self-portrait to create my final self-portrait. It is a very different
style compared to the original draft I had done, but it definitely pushes the limits of the programming language much
more and is ideally something I am more happier with.
